## 今後やりたいこと

今後やりたいことの軸は以下の 2 つです。

### 社会に大きな影響を与える、意義のある仕事
基本的に、仕事には必要に応じてなんでもやるというスタンスで取り組みます。
ただ、それは自身の仕事の意義をしっかり自覚していることに支えられています。
会社や事業の目指す方向を理解し、共感し、意義を持って仕事に取り組みたいと考えています。

### UXエンジニア
これまで必要に駆られてなんでもやるというスタンスで取り組んできました。
SketchやFigmaでのプロトタイプのデザインを作り、フルスタックに実装も行って、スマホアプリをリリースしたり、チームの拡大に合わせてEMなども経験しました。

それらの経験を踏まえると、私はチームの各個人が機能する環境にやりがいを感じ、自身は手を動かすことで貢献したいという思いが強いです。
具体的には、ユーザーに求められるものを探究すること、その為にユーザーに近い部分でフィードバックサイクルを回すことで、貢献できると思います。
